# Roadmap

## Next Major Areas

### Wayland Backend

Add honest backend support for Wayland-native input and screenshots where available, without pretending X11 semantics still apply.

### Accessibility-Tree Integration

Enable semantic targeting through accessibility APIs rather than only coordinates and window matching.

### Better OCR

Support alternate OCR engines, richer language handling, and better confidence reporting.

### Multi-Monitor Support

Add monitor-aware coordinate validation, targeting, and monitor enumeration.

### Semantic Element Targeting

Bridge screenshots, OCR, and future accessibility-tree data into higher-level target selection.

### Recording And Replay

Add reproducible capture/replay workflows for deterministic agent actions and debugging.

### Policy Plugins

Allow custom policy extensions for enterprise and lab environments.
